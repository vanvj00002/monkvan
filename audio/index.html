使用说明

  • 将音频文件（mp3、wav 等）上传到 static/audio_files/ 目录。
  • 页面会自动使用浏览器原生播放器，支持播放、暂停、下载。
  • 如需统一样式，可在主题 CSS 中添加 audio { max-width: 100%; margin: 1em 0; }。
